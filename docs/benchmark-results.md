## Metrics

- **Det%** — percentage of ground-truth tags correctly detected
- **RMSE** — root-mean-square corner error in pixels vs ground-truth
- **Max Error** — maximum corner error in pixels
- **FP** — false positive detections (tags detected that don't exist in the scene)
